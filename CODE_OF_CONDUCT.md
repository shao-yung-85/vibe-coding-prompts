# 行為準則 / Code of Conduct

## 我們的承諾 / Our Pledge

為了營造開放友善的環境，我們承諾讓參與這個專案的每個人，無論年齡、體型、身心障礙、族群、性別認同與表現、經驗程度、國籍、外貌、種族、宗教或性向，都能享有零騷擾的體驗。

We pledge to make participation in our project a harassment-free experience for everyone, regardless of background or identity.

## 我們的標準 / Our Standards

**鼓勵的行為 / Encouraged:**
- 友善、尊重、有建設性的溝通
- 對 prompt 提出具體、可操作的回饋
- 接受善意的批評，聚焦在「對社群最好」的做法

**不可接受的行為 / Unacceptable:**
- 騷擾、人身攻擊、歧視性言論
- 公開或私下的人身騷擾
- 未經同意公開他人私人資訊
- 其他在專業場合明顯不恰當的行為

## 範疇 / Scope

本準則適用於所有專案空間（Issue、PR、討論區），以及在公開場合代表本專案時。

This Code of Conduct applies within all project spaces and in public spaces when representing the project.

## 執行 / Enforcement

如遇到不當行為，請透過 GitHub Issue 或聯繫維護者回報。所有投訴都會被審視與調查，並做出適當且必要的回應。

Instances of unacceptable behavior may be reported by opening an issue or contacting the maintainer. All complaints will be reviewed and investigated.
