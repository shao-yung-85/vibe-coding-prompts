# 更新紀錄 / Changelog

本檔記錄這個 prompt 庫的重要變更。格式參考 [Keep a Changelog](https://keepachangelog.com/zh-TW/1.1.0/)。

All notable changes to this prompt library are documented here.

## [Unreleased]

### 新增 / Added
- ⚡ 速查表:12 句「情境 → 一句話 prompt」對照,放在 README 第一屏（中英文）
- 🆚 「爛 prompt vs 好 prompt」對照區:5 組 before/after 範例,展示背後共同原理（中英文）
- 🚨 翻車案例集 `examples/ai-gone-wrong.md`:真實 AI 亂改情境與救回來的 prompt

## [0.2.0] - 2026-06-14

### 新增 / Added
- 4 個新分類：🗄️ 資料庫 / SQL、🚢 部署 / DevOps、📝 寫文件、🧠 Prompt 通用心法（共擴充到 12 類、60 個 prompt）
- `README.en.md`：完整英文版（所有 prompt 本體英文化）
- `examples/`：實戰走查，示範如何把多個 prompt 串起來完成一個功能
- `.github/`：Issue 投稿表單、改善/回報表單、PR 模板（內建格式檢查清單）
- ⚡ 快速開始（30 秒上手）與 ⭐ Star CTA 區塊
- 徽章：prompts 數量、最後更新時間；語言切換列

### 變更 / Changed
- 目錄、CONTRIBUTING 分類清單、英文摘要同步加入新分類

## [0.1.0] - 2026-06-14

### 新增 / Added
- 初版發布：8 大分類、每類 5 個 prompt（專案初始化、新增功能、Debug、重構、寫測試、UI/樣式、護欄、CLAUDE.md 模板）
- 每個 prompt 附「為什麼有效」說明
- 工具對應表（Lovable / Bolt / Cursor / Claude Code）
- `CLAUDE-md-template.md`、`CONTRIBUTING.md`、MIT `LICENSE`
- 中英文雙語（中文為主、英文摘要）
